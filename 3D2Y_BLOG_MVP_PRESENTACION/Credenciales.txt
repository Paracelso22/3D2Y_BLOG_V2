🚀 COMANDOS PARA INICIAR EL PROYECTO:

1. Instalar dependencias:
   pip install -r requirements.txt

2. Aplicar migraciones:
   python manage.py migrate

3. Ejecutar servidor:
   python manage.py runserver

4. Abrir navegador en: http://127.0.0.1:8000/
